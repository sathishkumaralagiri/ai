# AI tool comparison for Agile practitioners

> A detailed side-by-side comparison of the three tools most commonly used by Scrum Masters and Agile Coaches.

## Quick reference

| | ChatGPT (GPT-4o) | Claude (Sonnet/Opus) | Microsoft Copilot |
|---|---|---|---|
| **Made by** | OpenAI | Anthropic | Microsoft / OpenAI |

| **Free tier** | Yes — GPT-4o mini | Yes — Claude Sonnet | Requires M365 licence |
| **Paid tier** | ChatGPT Plus (~$20/mo) | Claude Pro (~$20/mo) | M365 Copilot licence |
| **Context window** | ~128,000 tokens | ~200,000 tokens | ~128,000 tokens |
| **API available** | Yes | Yes | Limited |
| **Native integrations** | Plugins, GPTs | Claude.ai, API | Teams, Word, Excel, Outlook |
| **Consumer data training** | Opt-out available | Opt-out available | Not used for training |

## When to use each

### Use ChatGPT when:
- You need a highly structured table or formatted document
- You're doing data-heavy synthesis (velocity trends, metrics)
- You want to use custom GPTs or plugins
- You prefer the interface and are already familiar with it

### Use Claude when:
- You need nuanced, human-sounding facilitation language
- You're synthesising messy retrospective inputs
- You want to iterate on coaching approaches
- You have a long document or large paste of context

### Use Microsoft Copilot when:
- Your organisation uses M365 and has Copilot licences
- You want to summarise a Teams meeting automatically
- You're drafting directly in Word, PowerPoint, or Outlook
- You need enterprise-grade data privacy without opt-out management

## Privacy quick guide

| Scenario | Recommended tool |
|---|---|
| Testing with fictional data | Any — all are fine |
| Real team names, no PII | Any — use opt-out settings |
| Confidential business strategy | Enterprise tier only |
| Employee performance / personal info | Do not use AI at all |

---

## Recommendation for new users

Start with **Claude** (free at claude.ai). It handles the core Scrum Master tasks extremely well, has a generous free tier, and produces the most natural-sounding agile language. Once you're comfortable, add ChatGPT for structured outputs.

If your organisation deploys M365 Copilot, use it alongside — especially for Teams transcripts and document generation.
