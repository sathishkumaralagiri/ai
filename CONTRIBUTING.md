# Contributing to ai

Contributions are welcome — whether that's a new guide, a tool tip, a use case, or a correction.

## How to contribute

1. **Fork** this repository
2. **Create a branch** — `git checkout -b feature/your-topic`
3. **Write your content** — follow the structure of existing files
4. **Submit a pull request** — describe what you've added and why

## Content guidelines

- Write from real experience where possible — this is a practitioner resource, not a theory library
- Keep language direct and actionable
- Use the prompt code block format (triple backtick) for any prompts
- Don't paste AI-generated content without reviewing and editing it first
- Use role titles, not real names, in any examples

## What we're looking for

- New use case guides (e.g. AI for OKRs, AI for team health checks)
- Tool-specific tips (new tools, new features)
- Prompts that work well in practice
- Corrections to anything outdated

## Questions?

Open a GitHub issue or start a discussion.
